# Практические работы по системной и программной инженерии

Репозиторий содержит материалы по практическим занятиям 9 и 10 для проекта
`WinKey` - веб-приложения интернет-магазина цифровых товаров и лицензионных
ключей.

## Состав

- `practice_09/` - организация разработки: выбранная методология, GitHub и инструменты.
- `practice_10/` - документация разработки ПО: документация разработчика и пользователя.
- `reports/` - отчеты в формате Word для сдачи в университет.

## Проект

`WinKey` разрабатывается как e-commerce система на базе Next.js, TypeScript,
Prisma и PostgreSQL. В системе предусмотрены каталог товаров, корзина, оформление
заказа, интеграция с Digiseller, административная панель и механизм выдачи
цифрового товара после оплаты.
