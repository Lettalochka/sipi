# User Interface

## Назначение раздела

Этот раздел описывает основные элементы пользовательского интерфейса Legal Store.
Интерфейс предназначен для быстрого поиска цифровых товаров, оформления заказа и
получения результата покупки.

## Верхняя панель

В верхней панели доступны:

- переход в магазин;
- переключение языка интерфейса;
- открытие корзины;
- вход в учетную запись;
- создание нового аккаунта.

Верхняя панель остается главным навигационным элементом и помогает пользователю
быстро перейти к основным функциям.

## Главный экран

На главном экране пользователь видит:

- краткое описание магазина;
- строку поиска товаров;
- быстрые категории;
- блок преимуществ;
- подборки популярных и новых товаров.

Главный экран помогает начать покупку без перехода в дополнительные разделы.

## Карточка товара

Карточка товара содержит:

- изображение или визуальное обозначение товара;
- категорию;
- признак официальной поставки;
- цену;
- название товара;
- кнопку просмотра подробностей;
- кнопку добавления в корзину.

## Корзина

Интерфейс корзины показывает список выбранных товаров, количество, цену и итоговую
стоимость заказа. Из корзины пользователь переходит к оформлению.

## Страница заказа

Страница заказа показывает:

- номер заказа;
- текущий статус;
- информацию об оплате;
- результат выдачи цифрового товара;
- сообщение об ошибке, если выдача не была выполнена.

## Возможные проблемы

Пользователь может столкнуться со следующими ситуациями:

- товар не найден через поиск;
- товар временно недоступен;
- корзина пуста;
- оплата не подтверждена;
- выдача товара находится в обработке;
- заказ завершился с ошибкой.

## Результат

Интерфейс покрывает полный путь пользователя: от поиска товара до получения
цифрового ключа после оплаты.
