# Административная панель

Административная панель предназначена для управления товарами, категориями и
заказами.

## Управление товарами

Администратор может создавать, редактировать и отключать товары, настраивать
категории, цены, признаки популярности и параметры интеграции с Digiseller.

## Управление заказами

Администратор может просматривать заказы, отслеживать статусы выдачи и запускать
повторную обработку заказа при ошибке.

## Журнал выдачи

Для каждого процесса выдачи сохраняются события, которые помогают понять, на каком
этапе возникла проблема.
