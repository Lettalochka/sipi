# Просмотр заказа

Страница заказа показывает текущее состояние покупки и результат выдачи цифрового
товара.

## Статусы заказа

- `CREATED` - заказ создан;
- `PAID` - заказ оплачен;
- `FULFILLING` - выполняется выдача товара;
- `DELIVERED` - товар выдан;
- `FAILED` - при обработке возникла ошибка;
- `REFUNDED` - заказ возвращен.

## Доступ к заказу

Авторизованный пользователь может открыть собственные заказы. Для гостевого
заказа используется защищенный гостевой доступ.
