# Просмотр заказа

Страница заказа показывает текущее состояние покупки и результат выдачи цифрового
товара.

## HTML-код статуса заказа

```html
<section class="order-status">
  <h1>Order status</h1>
  <span class="status status-delivered">DELIVERED</span>
  <p>Your digital key is available on this page.</p>
</section>
```

После оплаты пользователь возвращается к странице заказа. Цифровой товар
становится доступен после подтверждения платежа.

## Статусы заказа

- `CREATED` - заказ создан;
- `PAID` - заказ оплачен;
- `FULFILLING` - выполняется выдача товара;
- `DELIVERED` - товар выдан;
- `FAILED` - при обработке возникла ошибка;
- `REFUNDED` - заказ возвращен.

## Доступ к заказу

Авторизованный пользователь может открыть собственные заказы. Для гостевого
заказа используется защищенный гостевой доступ.
