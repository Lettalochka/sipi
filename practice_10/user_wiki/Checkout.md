# Оформление заказа

Оформление заказа используется для перехода к оплате и фиксации выбранных товаров.

## Порядок оформления

1. Проверить состав корзины.
2. Перейти на страницу оформления заказа.
3. Подтвердить email для получения информации о заказе.
4. Выбрать оплату через Digiseller.
5. Перейти на платежную страницу.
6. После оплаты дождаться обработки заказа.

После успешной оплаты заказ получает статус обработки, затем цифровой товар
становится доступен на странице заказа.
