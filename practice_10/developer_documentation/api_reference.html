WinKey API Reference

Документация разработчика по серверным API проекта. Маршруты реализованы через Next.js Route Handlers в каталоге src/app/api.

Base URL: http://localhost:3000 Format: JSON Auth: NextAuth cookie Jobs auth: x-jobs-secret

Auth API

POST /api/auth/register Public, rate limited

Создает нового пользователя. Пароль хешируется перед сохранением.

Request

{
  "email": "user@example.com",
  "password": "password123",
  "name": "User"
}

Response 200

{
  "ok": true,
  "userId": "clx..."
}

Ошибки: 400 invalid, 409 email_taken, 429 rate_limited.

Checkout API

POST /api/checkout/create-payment Public/session, rate limited

Создает заказ и возвращает ссылку на оплату Digiseller.

Request

{
  "email": "buyer@example.com",
  "locale": "ru"
}

Response 200

{
  "ok": true,
  "orderId": "clx...",
  "redirectUrl": "https://oplata.info/..."
}

Favorites API

GET /api/favorites?ids=product_1,product_2 User session

Возвращает избранные товары пользователя или boolean-карту по переданным ID.

{
  "ok": true,
  "liked": {
    "product_1": true,
    "product_2": false
  }
}
POST /api/favorites/toggle User session

Добавляет товар в избранное или удаляет его из избранного.

{
  "productId": "product_id"
}
{
  "ok": true,
  "liked": true
}

Digiseller API

POST /api/digiseller/callback Signed webhook

Проверяет HMAC-подпись, сохраняет webhook-событие и переводит заказ в статус FULFILLING.

{
  "amount": "24.99",
  "currency": "USD",
  "invoice_id": "123456",
  "status": "paid",
  "signature": "hex_hmac_signature",
  "cf1": "order_id"
}

Signature string: amount:<amount>;currency:<currency>;invoice_id:<invoice_id>;status:<status>;

Jobs API

POST /api/jobs/fulfill Header x-jobs-secret

Запускает обработку ожидающих выдач цифрового товара.

curl -X POST http://localhost:3000/api/jobs/fulfill \
  -H "x-jobs-secret: $JOBS_SECRET" \
  -H "x-jobs-limit: 10"
POST /api/jobs/sync-digiseller Header x-jobs-secret

Синхронизирует цены или каталог Digiseller. Режим выбирается заголовком x-sync-mode: prices или catalog.

Locale API

POST /api/locale Public

Сохраняет выбранный язык интерфейса в cookie locale.

{
  "locale": "ru"
}
